# Lexical Analyzer

## Overview

This project implements a Lexical Analyzer in C that scans a C source program and classifies tokens such as keywords, identifiers, operators, literals, and special symbols. The project simulates the lexical analysis phase of a compiler using file handling and string processing techniques.

## Features

* Identifies C keywords
* Detects identifiers and literals
* Recognizes operators and special symbols
* Tokenizes source code efficiently
* Supports command line arguments
* Modular and structured implementation

## Technologies Used

* C Programming
* File Handling
* String Manipulation
* Command Line Arguments
* Modular Programming

### Working Principle

## Lexical Analysis Process
1. Read source code file
2. Scan character by character
3. Identify token patterns
4. Classify tokens into categories
5. Display tokenized output
